# Todo Api
[https://todoapi-1.herokuapp.com/todos](https://todoapi-1.herokuapp.com/todos)

## GET:

Endpoint | Function
--------- | ----------
/todos/ | List all Todos
/todos/:id/| List a single Todo

## POST:

Endpoint | Function
--------- | ---------
/todos/ | Create a Todo

## PATCH:

Endpoint | Function
--------- | ----------
/todos/:id | Update an existing todo

## DELETE:

Endpoint | Function
--------- | ----------
/todos/>id< | Delete an existing todo
